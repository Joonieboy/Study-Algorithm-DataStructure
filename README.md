# 알고리즘 & 자료구조 스터디

### 1. 목적
- 코딩테스트 준비

### 2. 언어
- C++

### 3. 스터디 방법
- 공통문제 2문제 선정 / 개인문제 1문제 선정
- 총 3문제 일주일동안 풀기
- 일주일 중 하루 온라인 미팅 (Google Meet)
- 온라인 미팅 시, 자기가 푼 문제 소개 & 풀이 방법 소개 -> 몰라서 못 푼 경우, 다같이 의논해서 풀기
- 안풀어오면 벌칙 -> 벌칙은 아직 미정
- 사전에 얘기하면 상관 X

### 4. 코드 관리 규칙
- 문제 풀 때마다 새 브랜치 생성
- 브랜치 이름은 {이니셜}/{문제이름or날짜or자유롭게}
- 온라인 미팅 후, main 브랜치에 머지
